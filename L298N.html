L298N - Driver de Motores

Descripción General

El L298N es un controlador de motores puente H dual que permite controlar dos motores DC o un motor paso a paso bipolar. Es muy utilizado en proyectos con Arduino y otros microcontroladores para controlar la dirección y velocidad de motores.

Características Técnicas

  • Tensión de alimentación: 5V a 35V.
  • Corriente máxima por canal: 2A (con disipador adecuado).
  • Control de velocidad mediante señales PWM.
  • Control de dirección mediante señales digitales.
  • Protección térmica integrada.
  • Incluye disipador de calor para evitar sobrecalentamiento.
  • Compatible con motores DC y motores paso a paso bipolares.
  • Interfaz sencilla para microcontroladores.
  • Voltaje lógico de control: 5V.
  • Dimensiones aproximadas del módulo: 43mm x 43mm.
  • Incluye bornes para conexión fácil de motores y alimentación.
  • Permite controlar dos motores de forma independiente.
  • Compatible con señales TTL para control digital.

Aplicaciones

Ideal para proyectos de robótica, automatización, vehículos controlados por Arduino, y cualquier sistema que requiera control de motores DC o motores paso a paso.
